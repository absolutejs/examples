Angular Convention Files

These files are detected automatically during the build and used as fallback UI when SSR fails.

  • error.ts Framework default error page for all Angular routes
  • not-found.ts 404 page when no Angular route matches

How It Works

When an Angular component throws during SSR, AbsoluteJS catches the error and calls the renderError function exported from error.ts. Angular convention files export functions that return HTML strings rather than Angular components.
