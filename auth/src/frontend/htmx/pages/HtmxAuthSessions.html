Sessions

Every active session for the signed-in user. The list endpoint is read-only; revoke buttons would need an HTMX fragment endpoint (currently returns JSON). The button below fetches the raw JSON response so you can inspect the shape. 

Click the button above to fetch /auth/sessions.

Per-session revoke would be DELETE /auth/sessions/:id. Wrapping that in an HTMX fragment endpoint (one that returns a re-rendered table on success) would be a package-level addition.
